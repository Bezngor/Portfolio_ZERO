Настройка переменных окружения для TextAgent:

1. Создайте файл .env в корне проекта
2. Добавьте в него следующие переменные:

# API ключ (от Anthropic или proxy сервиса)
ANTHROPIC_API_KEY=your-api-key-here

# По умолчанию используется proxyapi.ru, но можно указать другой URL:
# ANTHROPIC_BASE_URL=https://api.proxyapi.ru/anthropic

3. Или установите переменные окружения в системе:
   Windows: set ANTHROPIC_API_KEY=your-key-here
   Linux/Mac: export ANTHROPIC_API_KEY=your-key-here

Получить API ключ можно на сайте: https://console.anthropic.com/
